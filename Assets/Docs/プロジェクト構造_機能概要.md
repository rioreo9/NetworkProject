## ディレクトリ構成と機能のみ記載

### Assets/
プロジェクトのメインディレクトリ

```
Assets/
├── Animation/               # アニメーション素材
│   ├── Character/          # キャラクター用アニメーション
│   └── Object/             # オブジェクト用アニメーション
├── Docs/                   # プロジェクト文書
├── Editor/                 # エディター拡張
├── Material/               # マテリアル素材
├── Packages/               # パッケージ管理
├── Photon/                 # Photon Fusion2関連
├── Prefab/                 # プレハブ
├── Scene/                  # シーン
├── Scripts/                # スクリプト（メインロジック）
├── Settings/               # プロジェクト設定
├── TutorialInfo/           # チュートリアル情報
└── VisualPackage/          # ビジュアル素材
```

### Scripts/ - メインロジック

#### Core/ - コアシステム
```
Core/
├── DI/                     # 依存性注入システム
│   └── ProjectInstaller.cs # VContainer + VitalRouter統合
├── GameFlow/               # ゲームフロー管理
│   └── GameFlowHandler.cs  # ゲーム状態管理
├── Interfaces/             # インターフェース
│   └── IGameStateNotice.cs # ゲーム状態通知インターフェース
└── Wave/                   # ウェーブ管理システム
    ├── WaveHandler.cs      # ウェーブ進行管理
    ├── EnemyCoordinator.cs # 敵全体統括
    └── WaveConfiguration.cs # ウェーブ設定
```
